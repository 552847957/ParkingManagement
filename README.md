# 模块

模块名|命名空间|介绍
-|-|-
核心模块|Park.Core|提供数据库模型、业务处理方法
管理模块|Park.Admin|提供管理页面供管理员对各种信息进行管理、查看和汇总
用户模块|Park.Mobile|提供手机网站，供用户进行查看账户信息和停车场信息
外部接口模块|Park.API|为停车场硬件设施提供API
车位设计模块|Park.Designer|用于设计停车位地图。由于技术不够，故使用WPF作为设计器框架。
测试模块|Park.Test|用于测试，非单元测试
